Projects

Complete
AirDnD

This was a Software Engineering 2 class project whereby we had to build a fully working mobile and Web app, linked to the same server. I teamed up with 8 other students to deliver a minimum viable, fully functional app. The app is pretty much a dungeons and dragons character customization suite, with a database/ login and character update properties. An EC-2 instance with node.js was used to implement the database together with SQL. For the front end we used React and React-native for the purposes of time saving. React-native was chose for it's build once, deploy everywhere feature which allows for both I-OS and Android apps so share the same code base.


Politics Gallery

Initially it was a Graphics 1 project which was meant to simulate a simple 3D world with different camera angles and also allowing for movement and camera movement. I am guessing one bored Saturday I got bored and got too carried away and turned it into an entire politics gallery.
